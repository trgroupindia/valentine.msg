## Deployment Instructions for Vercel

1. Upload these files to your GitHub repository:
   - `index.html` (main website file)
   - `vercel.json` (Vercel configuration)

2. Connect your GitHub repository to Vercel

3. Deploy settings:
   - **Framework Preset**: Other
   - **Build Command**: Leave empty
   - **Output Directory**: Leave empty or use `.`
   - **Install Command**: Leave empty

4. Deploy!

## Features

- Fully animated Valentine's Day experience
- Interactive "Will You Be My Valentine" section
- Moving NO button (runs away when clicked!)
- Epic celebration when YES is clicked
- Particle effects, confetti, and fireworks
- Fully responsive design
- Custom cursor animations
